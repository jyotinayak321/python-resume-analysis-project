# Python Resume Analysis Project

## Project Overview

This project is an **Automatic Resume Checker** built using Python.
It analyzes a resume and checks whether important skills are present.

## Features

* Detects skills from resume
* Calculates resume score
* Shows matched and missing skills
* Simple text-based analysis

## Technologies Used

* Python
* NumPy
* Pandas

## Files

* `Automatic_Resume_Checker.py` → Main program
* `resume.txt` → Sample resume input

## How to Run

1. Install Python
2. Run the script

```
python Automatic_Resume_Checker.py
```

## Output

The program will show:

* Matched skills
* Missing skills
* Resume score
* Resume strength
